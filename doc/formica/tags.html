1 Помеченные функции

 (require formica/tags)

Библиотка formica/tags предоставляет инструменты для создания функций помеченных заданным символом.

Это позволяет отличать функции друг от друга и указывать на их свойства.

procedure

(tagged? f) → tagged?

  f : Fun
Возвращает #t, если x является помеченной функцией, и #f в противном случае.

procedure

(tag? f) → Sym

  f : tagged?
Возвращает метку помеченной функции f.

procedure

(check-tag f t) → Bool

  f : tagged?
  t : Any
Возвращает #t, если f является помеченной функцией, и имеет метку t, в противном случае – #f.

procedure

(set-tag t) → (-> Fun tagged?)

  t : Sym
(set-tag t n) → (Fun -> tagged?)
  t : Sym
  n : (or/c symbol? #f)
Возвращает оператор, помечающий функцию меткой t. Если указано имя n, то функция будет переименована.

procedure

(set-tag* t) → (-> Fun Fun)

  t : Sym
(set-tag* t n) → (Fun -> Fun)
  t : Sym
  n : (or/c symbol? #f)
Возвращает оператор, добавляющий к имени функции метку t. При этом переименованная функция не является помеченной. Если указано имя n, то функция будет переименована.

Примеры:
> (define t-cons ((set-tag 'tag) cons))
> t-cons

#<procedure:tag:cons>

> (tag t-cons)

'tag

> (check-tag 'tag t-cons)

#t

> (check-tag 'tag cons)

#f

> (tagged? t-cons)

#t

> (define r-cons ((set-tag* 'tag) cons))
> r-cons

#<procedure:tag:cons>

> (check-tag 'tag r-cons)

#f

> (tagged? r-cons)

#f
