7 Operators and functionals

 (require formica/tools)

The bindings documented in this section are provided by the formica/tools and formica modules.

procedure

(function? x) → Bool

  x : Any
Returns #t only if x is a function and #f otherwise.

procedure

(id x) → Any

  x : Any
The identity function.

Examples:
> (id 1)

1

> (id 'x)

'x

> (id +)

#<procedure:+>

> ((id +) 1 2)

3

> (id id)

#<procedure:id>

> (id '(a b c))

'(a b c)

procedure

(arg n) → Fun

  n : Ind
Creates a trivial function which returns the n-th argument.

Examples:
> (arg 1)

#<procedure:arg:1>

> ((arg 1) 'x 'y 'z)

'x

> ((arg 2) 'x 'y 'z)

'y

> ((arg 3) 'x 'y 'z)

'z

procedure

(const x) → Fun

  x : Any
Creates a trivial function which returns x for any arguments.

Examples:
> (const 'A)

#<procedure:const>

> ((const 'A) 1)

'A

> ((const 'A) 1 2)

'A

> ((const +))

#<procedure:+>

procedure

(composition f ...) → Fun

  f : Fun
Returns a generalized composition of functions f .... Function may have any arity: composition is done as if they all were curried.

In the generalized composition variadic or polyadic functions have minimal possible arity, unless they are greedy.

For function composition there is an alias: ∘ (could be entered as \circ + Alt \).

Examples:
> (define-formal (u 1) (b 2) (t 3))
> ((∘ u b) 1 2)

'(u (b 1 2))

> ((∘ b u) 1 2)

'(b (u 1) 2)

> ((∘ t b u) 1 2 3 4)

'(t (b (u 1) 2) 3 4)

> ((∘ t u b) 1 2 3 4)

'(t (u (b 1 2)) 3 4)

> ((∘ u t b) 1 2 3 4)

'(u (t (b 1 2) 3 4))

> ((∘ u b t) 1 2 3 4)

'(u (b (t 1 2 3) 4))

> ((∘ b u t) 1 2 3 4)

'(b (u (t 1 2 3)) 4)

> ((∘ b t u) 1 2 3 4)

'(b (t (u 1) 2 3) 4)

> ((∘ length (filter odd?)) '(1 2 3 4))

2

Composition with nullary function:
> (define-formal (n 0) (u 1) (b 2))
> ((∘ u n))

'(u (n))

> ((∘ b n) 1)

'(b (n) 1)

> ((∘ n u) 1)

nest: Cannot nest function with zero arity!

  function: #<procedure:formal:n>

  position: 2

Composition is associative:
> ((∘ (∘ b t) u) 1 2 3 4)

'(b (t (u 1) 2 3) 4)

> ((∘ b (∘ t u)) 1 2 3 4)

'(b (t (u 1) 2 3) 4)

Composition has left and right neutral element:
> ((∘ b id) 1 2)

'(b 1 2)

> ((∘ id b) 1 2)

'(b 1 2)

In the generalized composition variadic or polyadic functions have minimal possible arity:
> (define (f . x) (cons 'f x))
> (define (g x . y) (list* 'g x y))
> ((∘ f g) 1 2 3 4)

'(f (g 1) 2 3 4)

procedure

(greedy f) → Fun

  f : Fun
For variadic or polyadic function f returns equivalent function having maximal possible arity.

Examples:
> (define-formal f g h)
> ((∘ f g h) 1 2 3 4)

'(f (g (h 1)) 2 3 4)

> ((∘ f (greedy g) h) 1 2 3 4)

'(f (g (h 1) 2 3 4))

> ((∘ remove-duplicates (greedy append)) '(1 2 3) '(2 3 2 4))

'(1 2 3 4)

procedure

(negated p) → Fun

  p : Fun
Returns the negation of a predicate p.

Examples:
> (negated odd?)

#<procedure:negated:odd?>

> ((negated odd?) 2)

#t

> ((negated <) 1 2)

#f

procedure

(flipped f) → Fun

  f : Fun
Returns a function which is functionally equivalent to f, but gets it’s arguments in reversed order.

Examples:
> (define snoc (flipped cons))
> snoc

#<procedure:flipped:cons>

> (snoc 1 2)

'(2 . 1)

> ((flipped list) 1 2 3)

'(3 2 1)

procedure

(fif p f g) → Fun

  p : Fun
  f : Fun
  g : Fun
Returns function

x y ... ⟼ (if (p x y ...) (f x y ...) (g x y ...)).

Examples:
> (map (fif odd? sub1 id) '(1 2 3 4))

'(0 2 2 4)

procedure

(andf f g ...) → Fun

  f : Fun
  g : Fun
Returns function

x y ... ⟼ (and (f x y ...) (g x y ...) ...).

Examples:
> (map (andf integer? positive?) '(-3/5 -1 0 2 4.2))

'(#f #f #f #t #f)

procedure

(orf f g ...) → Fun

  f : Fun
  g : Fun
Returns function

x y ... ⟼ (or (f x y ...) (g x y ...) ...).

Examples:
> (map (orf integer? positive?) '(-3/5 -1 2 4.2))

'(#f #t #t #t)

procedure

(fork f g) → Fun

  f : Fun
  g : unary?
Returns function

x y ... ⟼ (f (g x) (g y) ...).

This function has an alias -<.

Examples:
> ((fork cons sqr) 2 3)

'(4 . 9)

> ((-< + sqr) 1 2 3)

14

procedure

(all-args p) → Fun

  p : Fun
Returns

(fork and p).

Examples:
> ((all-args real?) 2 -3 4.5)

#t

> ((all-args real?) 2 'x 4.5)

#f

procedure

(any-args f) → Fun

  f : Fun
Returns function

(fork or p).

Examples:
> ((any-args real?) '(1 2) 'a "abc" 1-2i)

#f

> ((any-args real?) 'x 2 "abc" 0+8i)

#t

procedure

(curry f arg ...) → (or/c curried? Any)

  f : Fun
  arg : Any
(curryr f arg ...) → (or/c curried? Any)
  f : Fun
  arg : Any
Return partially applied (curried) function f, with fixed arguments arg ....

Examples of partial application:
> (curry list 1 2)

#<procedure:curried:list>

> ((curry list 1 2) 3 4)

'(1 2 3 4)

> (map (curry cons 1) '(1 2 3))

'((1 . 1) (1 . 2) (1 . 3))

> (curryr list 1 2)

#<procedure:curried:list>

> ((curryr list 1 2) 3 4)

'(3 4 1 2)

> (map (curryr cons 1) '(1 2 3))

'((1 . 1) (2 . 1) (3 . 1))

> (curry cons 1 2)

'(1 . 2)

> (curryr cons 1 2)

'(1 . 2)

The curry function correctly reduces the arity of curried function:
> (procedure-arity cons)

2

> (procedure-arity (curry cons))

2

> (procedure-arity (curry cons 1))

1

> (procedure-arity (curryr cons 1))

1

> (procedure-arity (curry + 1 2 3))

(arity-at-least 0)

procedure

(curried? x) → boolean?

  x : Any
Returns #t if x is partially applied or curried function, and #f otherwise.

Examples:
> (curried? (curry cons 1))

#t

> (curried (curry +))

#<procedure:curried:+>

> (curried (curryr +))

#<procedure:curried:+>

> (curried? +)

#f

procedure

(fixed-point f [#:same-test same?]) → Fun

  f : Fun
  same? : (any/c any/c -> boolean?) = equal?
Returns a function

x ⟼ (f (f (f ... (f x))))

which finds a least fixed point of f by iterative application, while result keeps changing in the sense of the same? function.

If function f is not unary, it must return as many values, as it could accept.

Example:
> (define fcos (fixed-point cos))
> (fcos 1)

0.7390851332151607

> (cos (fcos 1))

0.7390851332151607

Finding a root of the equation F(x) = 0 by secant method:
> (define (root f)
    (compose
     I1
     (fixed-point (λ (x y)
                    (let ([fx (f x)]
                          [fy (f y)])
                      (values y (/ (- (* x fy) (* y fx))
                                   (- fy fx)))))
                  #:same-test almost-equal?)))
> ((root (λ (x) (- (* x x) 2))) 1.0 2.0)

1.414213562373095
